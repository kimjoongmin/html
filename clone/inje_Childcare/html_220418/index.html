홈페이지 구축 과정

상담부터 시안, 설계, 개발, 테스트에 걸친 전 과정을 전문가가 제작합니다.

  • 상담
  • 디자인 시안
  • 상세 설계
  • 개발/테스트
  • 홈페이지 오픈
  • 홈페이지 구축 시 기능 요구사항에 따라 2~3개월이 소요됩니다.
  • 상담 시 홈페이지 디자인과 사용할 기능(프로그램)을 협의합니다.
  • 홈페이지 오픈 후 유지보수를 지원합니다.

토탈 유지보수 서비스 제공

신속, 정확한 유지보수 서비스 제공을 약속합니다.

  • 팝업, 팝업존 제작행사, 교육 일정, 지원 정책 홍보 등 홈페이지
    공지가 필요한 부분에 대한 이미지 작업 지원
  • 콘텐츠 수정법, 제도 변경, 조직 변경 등
    변경사항 발생 시 콘텐츠 수정 지원
  • 기능 수정행사/교육 신청, 각종 서비스
    신청 관련 운영정책 변경 시 기능 수정 지원
  • 보안 점검 대응정기, 비정기 보안점검 결과
    취약점 발견 시 보완 조치
  • 홈페이지 호스팅홈페이지가 웹에서 정상적으로
    서비스될 수 있도록 호스팅 관리
  • 기타센터에서 홈페이지 운영관련
    요청하는 사항 대응
